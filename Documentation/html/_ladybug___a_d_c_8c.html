Detailed Description

Encapsulates the functions used to read an AIN on the nRF51822.

Author
Margaret Johnson
Version
1.0

The caller includes Ladybug_ADC.h and instantiates an extern variable e.g.: extern ADC_interface adc; then read - say AIN1 with int32_t adc_result_mV = adc.read(1); There is a Nordic white paper: "White paper content - nrf51 ADC.pdf" available on the Nordic web site.
