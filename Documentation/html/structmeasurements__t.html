Detailed Description

This structure is set up to hold the mV values read from the AINs used in measuring either the pH or EC. EC measurements use both an EC_Vin, and EC_Vout. This is why there are 2 int16's holding EC mV values. There is an int16 unused so an instance of this structure is word aligned.
