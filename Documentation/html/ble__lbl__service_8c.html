uint32_t ble_lbl_init ( ble_lbl_t *  p_lbl)

Get the S110 SoftDevice to know about the LBL Service and then add in the characteristics.

Function for initializing the LBL Service.

Note
The LBL's BLE initialization function is called within main.c to initialize the LBL's BLE service and characteristics...as well as initialize flash read/write variables used to store the pH calibration info (info goes into one of the characteristics)
Parameters
p_lblThe Ladybug Lite service structure defined to hold handles and other state information to the LBL service and it's characteristics.
Returns
either success or an error code
